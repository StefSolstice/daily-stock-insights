## ⚠️ 使用说明

### TuShare Token 配置

股票分析功能需要配置 TuShare token：

1. 获取 token: https://tushare.pro
2. 复制配置文件: `cp .env.example .env`
3. 编辑 `.env` 文件，填入 token:
   ```
   TUSHARE_TOKEN=your_token_here
   ```
4. 重启应用

**未配置 token 时**: 分析 API 会返回友好错误提示，不会崩溃。
